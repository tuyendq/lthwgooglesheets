# Learn the Hard Way - Google Sheets

## 4 common data types in Google Sheets
- Text: ISTEXT()  
- Number: ISNUMBER()  
- Date: ISDATE()  
- Logical: ISLOGICAL()  

- Check text is URL: ISURL()  
- Check formula: ISFORMULA()  
- Check Blank cell: ISBLANK()
